# EMAIL-DANE-MISMATCH

**DANE/TLSA records match the MX certificate**

| Field | Value |
|---|---|
| ID | `EMAIL-DANE-MISMATCH` |
| Family | `email` |
| Default severity | `high` |
| References | RFC 6698, RFC 7672 |

## Description

If TLSA records don't match the certificate presented by the MX server, DANE-aware senders will abort delivery. This can silently drop legitimate mail.
