# TLS-CIPHER-DH-WEAK

**DHE key exchange uses at least 2048-bit parameters**

| Field | Value |
|---|---|
| ID | `TLS-CIPHER-DH-WEAK` |
| Family | `tls` |
| Default severity | `high` |
| References | RFC 7919 |

## Description

DHE key exchange with primes smaller than 2048 bits is vulnerable to the Logjam attack (CVE-2015-4000), which allows a state-level adversary to downgrade and break the key exchange.
