# TLS-PROTOCOL-LEGACY-SSL3

**SSLv3 is disabled**

| Field | Value |

| Family | `tls` |

## Description

SSLv3 (1996) is vulnerable to the POODLE attack (CVE-2014-3566), which allows decryption of any chosen byte of the secret. It was formally deprecated by RFC 7568.
