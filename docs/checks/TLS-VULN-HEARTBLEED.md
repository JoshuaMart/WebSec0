## Description

Heartbleed (CVE-2014-0160) is a critical buffer over-read in OpenSSL's TLS heartbeat extension (RFC 6520). An attacker sends a heartbeat request with a payload_length far larger than the actual payload; vulnerable OpenSSL allocates a response buffer of payload_length bytes and copies that many bytes from heap memory, leaking private keys, session tokens, passwords, and other secrets. Affected versions: OpenSSL 1.0.1 through 1.0.1f.
