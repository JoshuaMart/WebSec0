# DNS-DNSSEC-WEAK-ALGO

**DNSSEC uses a modern signing algorithm**

| Field | Value |
|---|---|
| ID | `DNS-DNSSEC-WEAK-ALGO` |
| Family | `dns` |
| Default severity | `high` |
| References | RFC 8624 |

## Description

RFC 8624 forbids RSAMD5/DSA/RSASHA1; modern zones use ECDSA-P256 (alg 13) or Ed25519 (alg 15).
