# TLS-PROTOCOL-LEGACY-SSL2

**SSLv2 is disabled**

| Field | Value |

| ID | `TLS-PROTOCOL-LEGACY-SSL2` |
| Family | `tls` |
| Default severity | `critical` |

## Description

SSLv2 (1995) is fundamentally broken: it has no authentication of the server, weak export ciphers, and trivially-crackable key exchange. Any server still accepting SSLv2 connections is critically compromised.
