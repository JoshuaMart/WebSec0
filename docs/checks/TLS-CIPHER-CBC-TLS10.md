# TLS-CIPHER-CBC-TLS10

**CBC ciphers not accepted under TLS 1.0**

| Field | Value |
|---|---|
| ID | `TLS-CIPHER-CBC-TLS10` |
| Family | `tls` |
| Default severity | `medium` |
| References | RFC 8996 |

## Description

CBC cipher suites in TLS 1.0 are vulnerable to the BEAST attack (CVE-2011-3389), which allows a man-in-the-middle to recover plaintext bytes. Disabling TLS 1.0 fully mitigates this.
