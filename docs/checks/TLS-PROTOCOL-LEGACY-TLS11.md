# TLS-PROTOCOL-LEGACY-TLS11

## Description

TLS 1.1 was deprecated by RFC 8996 (March 2021). While less vulnerable than TLS 1.0, it still lacks the security improvements of TLS 1.2/1.3 and is no longer compliant with modern standards.
