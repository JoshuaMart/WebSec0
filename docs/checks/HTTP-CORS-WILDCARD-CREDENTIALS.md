# HTTP-CORS-WILDCARD-CREDENTIALS

**CORS does not combine wildcard origin with credentials**

| Field | Value |
|---|---|
| ID | `HTTP-CORS-WILDCARD-CREDENTIALS` |
| Family | `http` |
| Default severity | `high` |
| References | WHATWG Fetch §3.2 |

## Description

Access-Control-Allow-Origin: * with Access-Control-Allow-Credentials: true is forbidden by the spec; servers shouldn't ship this combination.
