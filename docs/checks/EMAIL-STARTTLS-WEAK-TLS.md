# EMAIL-STARTTLS-WEAK-TLS

**STARTTLS negotiates TLS 1.2 or higher**

| Field | Value |
|---|---|
| ID | `EMAIL-STARTTLS-WEAK-TLS` |
| Family | `email` |
| Default severity | `medium` |
| References | RFC 3207, RFC 8996 |

## Description

TLS 1.0 and 1.1 were deprecated by RFC 8996 (March 2021). SMTP STARTTLS should use TLS 1.2 as the minimum to protect mail transit.
