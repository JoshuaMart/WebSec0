# TLS-PROTOCOL-LEGACY-TLS10

**TLS 1.0 is disabled**

| Field | Value |
|---|---|
| ID | `TLS-PROTOCOL-LEGACY-TLS10` |
| Family | `tls` |
| Default severity | `high` |

## Description

TLS 1.0 was deprecated by RFC 8996 (March 2021). It is vulnerable to the BEAST attack when used with CBC ciphers, and is no longer compliant with PCI-DSS 4.0 or NIST SP 800-52 Rev. 2.
