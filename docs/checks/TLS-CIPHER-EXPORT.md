# TLS-CIPHER-EXPORT

**EXPORT cipher suites are rejected**

| Field | Value |
|---|---|
| ID | `TLS-CIPHER-EXPORT` |
| Family | `tls` |
| Default severity | `critical` |
| References | RFC 7465 |

## Description

EXPORT cipher suites use artificially weakened key material (40-bit keys) mandated by 1990s US export regulations. They are the basis of the FREAK attack (CVE-2015-0204).
