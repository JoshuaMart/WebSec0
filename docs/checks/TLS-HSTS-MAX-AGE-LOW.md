# TLS-HSTS-MAX-AGE-LOW

**HSTS max-age is at least one year**

| Field | Value |
|---|---|
| ID | `TLS-HSTS-MAX-AGE-LOW` |
| Family | `tls` |
| Default severity | `medium` |
| References | RFC 6797 |

## Description

Browsers expect max-age ≥ 31536000 to honour the policy long-term and to qualify for preload.
