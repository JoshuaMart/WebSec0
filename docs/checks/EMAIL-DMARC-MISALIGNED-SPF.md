# EMAIL-DMARC-MISALIGNED-SPF

**DMARC SPF alignment is strict**

## Description

`aspf=s` requires the envelope sender domain to match the From domain exactly. The default `aspf=r` (relaxed) allows subdomain matches, which is weaker.
