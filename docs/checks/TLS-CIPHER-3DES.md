# TLS-CIPHER-3DES

**3DES cipher suites are rejected**

| Field | Value |
|---|---|
| ID | `TLS-CIPHER-3DES` |
| Family | `tls` |
| Default severity | `medium` |
| References | RFC 9325 |

## Description

3DES is vulnerable to the Sweet32 birthday attack (CVE-2016-2183) after ~785 GB of ciphertext due to its 64-bit block size. NIST deprecated 3DES in 2017.
