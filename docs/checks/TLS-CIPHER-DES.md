# TLS-CIPHER-DES

**DES cipher suites are rejected**

| Field | Value |
|---|---|
| ID | `TLS-CIPHER-DES` |
| Family | `tls` |
| Default severity | `high` |
| References | RFC 9325 |

## Description

Single DES uses 56-bit effective key length, exhaustively broken by EFF's DES Cracker in 1998. Any ciphertext encrypted with DES can be decrypted in under 24 hours today.
