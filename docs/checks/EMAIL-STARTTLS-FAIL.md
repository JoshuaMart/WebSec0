# EMAIL-STARTTLS-FAIL

**MX server advertises STARTTLS**

| Default severity | `high` |

## Description

STARTTLS (RFC 3207) allows MTAs to negotiate TLS before delivering mail. Without it, all email transit to this domain is in cleartext and interceptable.
