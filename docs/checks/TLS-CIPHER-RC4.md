# TLS-CIPHER-RC4

**RC4 cipher suites are rejected**

| Field | Value |
|---|---|
| ID | `TLS-CIPHER-RC4` |
| Family | `tls` |
| Default severity | `high` |
| References | RFC 7465 |

## Description

RC4 has multiple cryptographic biases that allow session key recovery given sufficient ciphertext. It was prohibited by RFC 7465 in February 2015.
