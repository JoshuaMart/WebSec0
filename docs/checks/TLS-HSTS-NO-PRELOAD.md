## Description

The HSTS `preload` directive, combined with `includeSubDomains` and `max-age ≥ 10886400`, makes the domain eligible for inclusion in browser HSTS preload lists, providing protection even for first-time visitors. Verification of actual list inclusion requires checking hstspreload.org (deferred).
