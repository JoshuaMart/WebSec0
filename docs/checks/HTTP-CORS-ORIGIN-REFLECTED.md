# HTTP-CORS-ORIGIN-REFLECTED

**CORS does not reflect arbitrary origins**

| Field | Value |
|---|---|
| ID | `HTTP-CORS-ORIGIN-REFLECTED` |
| Family | `http` |
| Default severity | `high` |
| References | WHATWG Fetch §3.2 |

## Description

Reflecting the request Origin in Access-Control-Allow-Origin lets any host become trusted — implement a strict allowlist instead.
