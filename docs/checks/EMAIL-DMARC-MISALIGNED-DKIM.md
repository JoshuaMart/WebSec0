# EMAIL-DMARC-MISALIGNED-DKIM

**DMARC DKIM alignment is strict**

## Description

`adkim=s` requires the DKIM `d=` signing domain to exactly match the From header domain. The default `adkim=r` (relaxed) allows subdomain matches.
